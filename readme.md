# Mintlify-Inspired Landing Page (HTML + CSS Only)

 — built using **only HTML and CSS** (no JavaScript, no frameworks).

---

## Features

- structured navbar with logo, links, and CTA buttons
- Hero section with announcement badge and email CTA input
- Hero preview image section
- Brand/logo grid section (SVG logos)
- Feature sections (cards with GIFs and content)
- Customer story card with stats
- Enterprise highlight section with CTA button
- Customer cards section (images + centered logo overlay)
- Footer with multi-column navigation links and security badge area

---

## Tech Stack

- **HTML5**
- **CSS3**
- Google Fonts: **Inter**, **Montserrat**
- SVG icons/logos embedded directly in HTML

## How to Run Locally

### Option 1: Open directly
1. Download/clone the project
2. Open `index.html` in your browser

### Option 2: VS Code Live Server (recommended)
1. Open the project folder in VS Code
2. Install **Live Server**
3. Right-click `index.html` → **Open with Live Server**
